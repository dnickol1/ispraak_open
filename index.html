iSpraak for Google Chrome is an online activity generator that automates speech evaluation for second language learners. Create an activity below, or manage existing activities via the ☰ menu button.

  • Click the menu ☰ button to login or explore existing activity sets.

  • Select your language of instruction. iSpraak now supports 36 languages!

  • Leave this as the default to have iSpraak generate a new text-to-speech (TTS) audio file for you.

  • Very short contextualized texts work best for speech recognition and speech synthesis.
